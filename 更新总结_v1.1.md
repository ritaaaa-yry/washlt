## 🎯 核心技术升级

### 图像处理技术

| 技术 | 用途 | OpenCV 函数 |
|-----|------|------------|
| 直方图均衡化 | 增强对比度 | `cv2.equalizeHist()` |
| 高斯模糊 | 降噪 | `cv2.GaussianBlur()` |
| 自适应阈值 | 处理光照不均 | `cv2.adaptiveThreshold()` |
| OTSU 阈值 | 自动确定阈值 | `cv2.THRESH_OTSU` |
| 形态学开运算 | 去除噪点 | `cv2.MORPH_OPEN` |
| 形态学闭运算 | 填充空洞 | `cv2.MORPH_CLOSE` |
| 轮廓检测 | 找到污渍区域 | `cv2.findContours()` |
| 轮廓绘制 | 标记污渍 | `cv2.drawContours()` |

### 算法流程

```python
# 污渍检测核心算法
def detect_stains(gray_image):
    # 1. OTSU 自动阈值
    _, otsu_binary = cv2.threshold(
        gray_image, 0, 255,
        cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU
    )
    
    # 2. 形态学开运算 - 去除小噪点
    kernel = np.ones((3, 3), np.uint8)
    opening = cv2.morphologyEx(
        otsu_binary, cv2.MORPH_OPEN, 
        kernel, iterations=2
    )
    
    # 3. 形态学闭运算 - 填充小空洞
    stain_mask = cv2.morphologyEx(
        opening, cv2.MORPH_CLOSE, 
        kernel, iterations=2
    )
    
    return stain_mask
```
