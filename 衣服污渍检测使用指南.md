## 🔧 参数调整指南

如果检测效果不理想，可以调整以下参数：

### 1. 阈值参数（最常调整）

**位置**：`src/demo.py` 第 507 行附近

```python
# 当前设置
binary_image = threshold_image(denoised_image, threshold_value=127)
```

**调整建议**：
- **检测不到污渍**：降低阈值 → 改为 `100` 或 `80`
- **误报太多**：提高阈值 → 改为 `150` 或 `180`
- **范围**：0-255

### 2. 边缘检测参数

**位置**：`src/demo.py` 第 513 行附近

```python
# 当前设置
edges = edge_detection(denoised_image, low_threshold=50, high_threshold=150)
```

**调整建议**：
- **边缘太少**：降低阈值 → `low_threshold=30, high_threshold=100`
- **边缘太多（噪声）**：提高阈值 → `low_threshold=70, high_threshold=200`

### 3. 降噪强度

**位置**：`src/demo.py` 第 97 行附近

```python
# 当前设置
denoised = cv2.GaussianBlur(gray_image, (5, 5), 0)
```

**调整建议**：
- **降噪更强**：改为 `(7, 7)` 或 `(9, 9)`
- **保留更多细节**：改为 `(3, 3)`
- **核大小必须是奇数**

### 4. 形态学操作参数

**位置**：`src/demo.py` 第 188-194 行

```python
# 当前设置
kernel = np.ones((3, 3), np.uint8)
opening = cv2.morphologyEx(otsu_binary, cv2.MORPH_OPEN, kernel, iterations=2)
stain_mask = cv2.morphologyEx(opening, cv2.MORPH_CLOSE, kernel, iterations=2)
```

**调整建议**：
- **去除更多噪点**：增加 `iterations` → `iterations=3` 或 `4`
- **保留小污渍**：减少 `iterations` → `iterations=1`
- **更强的效果**：增大核 → `kernel = np.ones((5, 5), np.uint8)`

## 💡 实用技巧

### 技巧 1：快速测试参数

1. 使用**模式 1**处理单张图像
2. 修改参数
3. 重新运行
4. 观察效果变化
5. 找到最佳参数后，再用**模式 2**批量处理

### 技巧 2：对比不同图像

```bash
# 处理第 1 张图像
python src/demo.py
选择: 1
图像编号: 1

# 处理第 20 张图像
python src/demo.py
选择: 1
图像编号: 20

# 对比 output/ 文件夹中的结果
```

### 技巧 3：分批处理

如果 85 张图像太多，可以修改代码分批处理：

在 `demo.py` 的 `main()` 函数中，找到：
```python
batch_process_images(clothes_dir, max_images=10, show_display=False)
```

修改 `max_images` 的值：
- 前 20 张：`max_images=20`
- 前 50 张：`max_images=50`
- 所有图像：`max_images=None`

### 技巧 4：清理输出文件夹

```bash
# Windows: 删除 output 文件夹中的所有文件
del /q output\*.*

# 或者手动删除 output 文件夹中的文件
```

## ❓ 常见问题

### Q1: 处理 85 张图像需要多长时间？

**A:** 取决于你的电脑性能：
- **测试模式（10 张）**：1-2 分钟
- **全部 85 张**：8-15 分钟
- **单张图像（显示窗口）**：20-30 秒

### Q2: 检测到的污渍不准确怎么办？

**A:** 尝试调整参数：
1. 先用**模式 1**处理几张图像
2. 观察哪些步骤出现问题
3. 根据"参数调整指南"修改参数
4. 重新测试直到效果满意
5. 用**模式 2**批量处理

### Q3: 可以只处理某几张图像吗？

**A:** 可以，修改代码：

方法 1：在 `main()` 函数中指定图像列表
```python
# 处理 1.jpg, 5.jpg, 10.jpg
image_files = [
    Path('data/Clothes_Samples/1.jpg'),
    Path('data/Clothes_Samples/5.jpg'),
    Path('data/Clothes_Samples/10.jpg'),
]
```

方法 2：使用切片
```python
# 处理 20-30 张图像
image_files = sorted(clothes_dir.glob('*.jpg'))[19:30]
```

### Q4: 内存不足怎么办？

**A:** 
- 使用**模式 3**（测试模式）
- 分批处理（每次处理 10-20 张）
- 处理完一批后清空 `output/` 文件夹
- 不要同时显示所有图像（使用批量模式）

### Q5: 如何保存检测统计数据？

**A:** 修改 `batch_process_images()` 函数，添加日志记录：

```python
# 在函数末尾添加
import csv
with open('output/statistics.csv', 'w', newline='') as f:
    writer = csv.writer(f)
    writer.writerow(['Image', 'Stain_Count'])
    # 记录每张图像的数据
```
