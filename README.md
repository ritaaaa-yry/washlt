# 衣服污渍检测项目 (Clothes Stain Detection)

基于 Python 和 OpenCV 的衣服污渍检测系统，用于自动检测和标记衣服上的污渍。

## 📋 项目概述

本项目实现了一个完整的衣服污渍检测系统，包括：
- 图像加载与预处理
- 对比度增强和降噪
- 多种阈值处理方法
- 边缘检测
- 污渍区域检测和标记
- 批量处理功能
- 处理结果的可视化与保存

## 📖 功能说明

### demo.py 实现的功能

#### 1. 基础图像处理

| 函数名 | 功能 | 说明 |
|-------|------|------|
| `load_image()` | 加载图像 | 从指定路径读取衣服图像 |
| `convert_to_grayscale()` | 灰度化 | 将彩色图像转为灰度图 |
| `enhance_contrast()` | 对比度增强 | 使用直方图均衡化增强对比度 |
| `denoise_image()` | 降噪处理 | 使用高斯模糊去除噪声 |

#### 2. 阈值处理

| 函数名 | 功能 | 说明 |
|-------|------|------|
| `threshold_image()` | 固定阈值二值化 | 使用 `cv2.threshold()` |
| `adaptive_threshold()` | 自适应阈值 | 适应光照不均的情况 |

#### 3. 污渍检测

| 函数名 | 功能 | 说明 |
|-------|------|------|
| `edge_detection()` | 边缘检测 | 使用 Canny 算法检测边缘 |
| `detect_stains()` | 污渍检测 | 使用 OTSU 阈值和形态学操作 |
| `mark_stains_on_original()` | 标记污渍 | 在原图上用红色标记污渍区域 |

#### 4. 批量处理

| 函数名 | 功能 | 说明 |
|-------|------|------|
| `process_single_image()` | 处理单张图像 | 完整的处理流程 |
| `batch_process_images()` | 批量处理 | 处理整个文件夹的图像 |

### 处理流程

```
原始图像
    ↓
灰度化处理
    ↓
对比度增强（直方图均衡化）
    ↓
降噪处理（高斯模糊）
    ↓
┌─────────────┬─────────────┬─────────────┐
│固定阈值二值化│自适应阈值处理│边缘检测(Canny)│
└─────────────┴─────────────┴─────────────┘
    ↓
污渍检测（OTSU + 形态学操作）
    ↓
在原图上标记污渍区域
    ↓
保存结果到 output/ 文件夹
```

## 🔧 参数调整

你可以在 `src/demo.py` 中调整以下参数来优化检测效果：

### 阈值参数

```python
# 固定阈值（第 119 行附近）
binary_image = threshold_image(denoised_image, threshold_value=127)
# 调整范围：0-255，值越低检测越敏感

# 边缘检测阈值（第 125 行附近）
edges = edge_detection(denoised_image, low_threshold=50, high_threshold=150)
# 调整 low_threshold 和 high_threshold 来改变边缘检测灵敏度
```

### 降噪参数

```python
# 高斯模糊核大小（第 97 行附近）
denoised = cv2.GaussianBlur(gray_image, (5, 5), 0)
# 将 (5, 5) 改为 (3, 3) 或 (7, 7) 来调整降噪强度
```

### 形态学操作参数

```python
# 形态学核大小（第 188 行附近）
kernel = np.ones((3, 3), np.uint8)
# 调整为 (5, 5) 或 (7, 7) 来改变操作强度

# 开运算迭代次数
opening = cv2.morphologyEx(otsu_binary, cv2.MORPH_OPEN, kernel, iterations=2)
# 增加 iterations 可以去除更多小噪点
```

## 🔍 常见问题

### Q1: 找不到 Clothes_Samples 文件夹？

**A:** 确保图像文件夹位于 `data/Clothes_Samples/` 路径下。

```bash
dachuang/
└── data/
    └── Clothes_Samples/
        ├── 1.jpg
        ├── 2.jpg
        └── ...
```

### Q2: 检测到太多误报（假阳性）？

**A:** 可以尝试：
1. 提高固定阈值（如从 127 改为 150）
2. 增加形态学操作的迭代次数
3. 调整边缘检测的阈值

### Q3: 检测不到污渍？

**A:** 可以尝试：
1. 降低固定阈值（如从 127 改为 100）
2. 检查图像质量是否足够
3. 调整对比度增强参数

### Q4: 处理速度太慢？

**A:** 
- 使用批量处理模式（选项 2 或 3）
- 将 `show_display` 参数设为 `False`
- 减少处理的图像数量

### Q5: 如何添加自己的图像？

**A:** 
1. 将图像文件（.jpg 格式）放入 `data/Clothes_Samples/` 文件夹
2. 运行 `python src/demo.py`
3. 选择相应的处理模式

## 📚 技术说明

### 使用的图像处理技术

1. **灰度化**：`cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)`
2. **直方图均衡化**：`cv2.equalizeHist(gray)` - 增强对比度
3. **高斯模糊**：`cv2.GaussianBlur()` - 降噪
4. **固定阈值**：`cv2.threshold()` - 二值化
5. **自适应阈值**：`cv2.adaptiveThreshold()` - 处理光照不均
6. **Canny 边缘检测**：`cv2.Canny()` - 检测边缘
7. **OTSU 阈值**：`cv2.THRESH_OTSU` - 自动确定最佳阈值
8. **形态学操作**：`cv2.morphologyEx()` - 开运算和闭运算
9. **轮廓检测**：`cv2.findContours()` - 找到污渍区域

### 算法流程

```
输入图像
    ↓
[预处理] 灰度化 → 增强对比度 → 降噪
    ↓
[阈值处理] 固定阈值 + 自适应阈值 + 边缘检测
    ↓
[污渍检测] OTSU阈值 → 形态学开运算 → 形态学闭运算
    ↓
[结果标记] 轮廓检测 → 在原图上绘制红色轮廓
    ↓
输出结果
```
